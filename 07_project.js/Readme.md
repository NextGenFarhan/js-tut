# JavaScript Mini Projects

This repository contains **four mini JavaScript projects** created to strengthen fundamental JavaScript concepts through practical implementation.

Each project focuses on improving core skills such as DOM manipulation, event handling, logical thinking, and basic JavaScript functionality.

---

## Projects Included

### 1. Background Color Changer
- Contains four boxes with unique colors.
- Clicking on any box changes the background color of the page to the selected color.
- Demonstrates DOM manipulation and click events.

### 2. BMI Calculator
- Calculates the Body Mass Index (BMI) based on user input.
- Displays the BMI result.
- Helps understand form handling and basic calculations.

### 3. Digital Clock
- Displays the current time.
- Updates automatically in real-time.
- Uses the JavaScript Date object.

### 4. Guess the Number Game
- Generates a random number between 1 and 100.
- The user tries to guess the correct number.
- Provides feedback based on the user's guess.
- Covers random number generation and conditional logic.

---

## Purpose

The main aim of this repository is to:
    - Strengthen JavaScript basic concepts
    - Practice logic building
    - Improve understanding of DOM manipulation and events
